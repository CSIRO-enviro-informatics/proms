Endpoints

Name Description Access
Provenance object Accesses a single object from the provenance database

{{ web_subfolder }}/object?url={uri}

uri (mandatory): URI of an object in the provenance database
Provenance object register Accesses a register (index) of one class of object from the provenance database

{{ web_subfolder }}/register/{class_name}/

class_name (mandatory): the name of a class in the provenance database
class_names available are:
  • Report
  • ReportingSystem
  • Activity
  • Agent
  • Entity
  • Person
SPARQL endpoint Provides a read-only SPARQL 1.1 endpoint for the provenance database

{{ web_subfolder }}/function/sparql

See the SPARQL endpoint itself for documentation of use
Register a ReportingSystem Manually register a ReportingSystem for which Reports can then be sent to this PROMS Server instance

{{ web_subfolder }}/function/register_reportingsystem
Create a Report Manually create a PROMS Report to download or send to this PROMS Server instance using a web form

{{ web_subfolder }}/function/create_report
New Report POST a Report to this PROMS Server instance

{{ web_subfolder }}/function/new_report

Must be an HTTP POST request containing a valid PROMS Report for a registered ReportingSystem in one of the following RDF mimetypes: 'text/turtle', 'application/rdf+xml', 'application/rdf+json', 'application/json', 'text/ntriples', 'text/nt', 'text/n3'
