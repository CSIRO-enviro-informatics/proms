PROMS Server

For the EUDM project

This is the home page for an instance of PROMS (PROvenance Management System) for the EUDM project. It contains a database to store provenance data and a series of functions to manage that data.

The EUDM project tracks the provenance of data that is incorporated into the energy use data model and that provenance is stored within this system.
